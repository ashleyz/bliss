 14      '''SAGA base object as defined in GFD.90  
 15   
 16      As Python natively provides most functionality of the SAGA base object class 
 17      as specified in GFD.90, this object's only function is to support the use of 
 18      L{Session}s.  In particular, this class allows to obtain the session any 
 19      object is living in, to re-use that for other objects and operations. 
 20   
 21      Example:: 
 22   
 23          # run a job service in a session 
 24          c = saga.Context('X509') 
 25          c['UserCert'] = '/tmp/x509_special.pem' 
 26   
 27          s = saga.Session() 
 28          s.add_context(c) 
 29   
 30          js = saga.job.Service(s) 
 31          j  = js.create_job(...)   # the job inherits the js' session! 
 32          j.run() 
 33   
 34          ... 
 35   
 36          # somewhere else in the code, you want to stage an output file for the 
 37          # job, once it finished.  To use the same credentials (which presumably 
 38          # worked for the job's backend), we obtain the job's session, and re-use 
 39          # it for file staging. 
 40          j.wait() 
 41          s = j.get_session() 
 42          d = j.get_description()['WorkingDirectory'] 
 43           
 44          remote = saga.filesystem.Directory(s, d) 
 45          remote.copy('output', 'file://localhost/tmp/output')
