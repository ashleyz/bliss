SAGA - A Simple API for Grid Applications¶

Introduction¶

SAGA-Bliss is a light-weight Python package that implements parts of the OGF GFD.90 SAGA interface specification and provides plug-ins for different distributed middleware systems and services. SAGA-Bliss implements the most commonly used features of GFD.90 based upon extensive use-case analysis, and focuses on usability and simple deployment in real-world heterogeneous distributed computing environments and application scenarios.

More information can be found at http://saga-project.github.com/bliss
