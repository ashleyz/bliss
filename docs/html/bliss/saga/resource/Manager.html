Manager¶

class bliss.saga.resource.Manager(url, session=None)[source]¶

The resource manager class, as the name suggests, manages resource instances. It may be responsible for managing the lifetime of such instances ( bliss.saga.create / bliss.saga.destroy / bliss.saga.get ), and for inspecting them ( bliss.saga.list / bliss.saga.list_templates / bliss.saga.get_template_details ) [method links are for compute resources]:

# obtain a handle to a suitable resource, and wait until it is active
rm = saga.resource.Manager(url)
cr = rm.create_compute(cd)
cr.wait(saga.resource.State.Active)

The resources managed by the manager can have different types – they can be storage or compute resources. For the respective differences and the resource description details, see

  • bliss.saga.resource.Storage
  • bliss.saga.resource.StorageDescription
  • bliss.saga.resource.Compute
  • bliss.saga.resource.ComputeDescription
