Bliss, a SAGA implementation in Python¶

Bliss (Bliss IS SAGA) is a pragmatic, light-weight implementation of the OGF SAGA standard (GFD.90). Bliss is written 100% in Python and focuses on usability and ease of deployment rather than on feature completeness or strict standard obedience.

Please refer to the Bliss wiki for installation instructions, user guide, etc:
