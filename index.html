Bliss

A Pragmatic SAGA Implementation for Python

  • Bliss (BLiss IS Saga) is partial and rather 'laissez-faire' implementation of the OGF GFD.90 SAGA standard. Bliss is written 100% in Python and focuses on usability and ease of deployment rather than feature completeness or blind standard obedience.

    SAGA defines a high-level API for uniform access to the building-blocks of distributed computing: (1) job execution and managegment, (2) file and data management and (3) distributed coordination and orchestration. Bliss is a native Python implementation of this API.

  • Bliss installs in seconds with Python pip:
    Installation instructions

    $ pip install bliss
