Class Manager

source code

The resource manager class, as the name suggests, manages resource instances. It may be responsible for managing the lifetime of such instances ( create / destroy / get ), and for inspecting them ( list / list_templates / get_template_details ) [method links are for compute resources]:

 # obtain a handle to a suitable resource, and wait until it is active
 rm = saga.resource.Manager(url)
 cr = rm.create_compute(cd)
 cr.wait(saga.resource.State.Active)

The resources managed by the manager can have different types -- they can be storage or compute resources. For the respective differences and the resource description details, see

  • resource.Storage
  • resource.StorageDescription
  • resource.Compute
  • resource.ComputeDescription
