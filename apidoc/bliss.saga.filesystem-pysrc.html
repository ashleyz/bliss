 4  '''SAGA Filesystem Package API. 
 5   
 6  The SAGA filesystem package provides the ability to interact with (local and 
 7  remote) file systems.  The two provided classes, L{filesystem.Directory} and 
 8  L{filesystem.File} represent the well known abstractions common to all different 
 9  types of file systems.  The API provides a number of operations, which all 
10  behave like the common unix command line tools (copy, list, remove etc):: 
11   
12      # get a directory handle 
13      dir = saga.filesystem.Directory("sftp://localhost/tmp/") 
14   
15      # create a subdir 
16      dir.make_dir ("data/") 
17   
18      # list contents of the directory 
19      files = dir.list () 
20   
21      # copy *.dat files into the subdir 
22      for f in files : 
23          if re.match ('^.*\.dat$', f) : 
24              dir.copy (f, "sftp://localhost/tmp/data/") 
25   
26   
27   
28  The above example covers most of the semantics of the filesystem package -- 
29  additional capabilities, such get_size() or move(), can be found in the 
30  individual class documentations. 
31  '''
