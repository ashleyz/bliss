 4  ''' B{File/Directory Management} 
 5   
 6      The file managment API provides the ability to interact with (local and 
 7      remote) file systems via the two classes, L{filesystem.Directory} and 
 8      L{filesystem.File}. The API provides a number of operations, which all 
 9      behave similar to the common unix command line tools (cp, ls, rm etc). 
10   
11      B{Example}:: 
12   
13        # get a directory handle 
14        dir = saga.filesystem.Directory("sftp://localhost/tmp/") 
15   
16        # create a subdir 
17        dir.make_dir ("data/") 
18   
19        # list contents of the directory 
20        files = dir.list () 
21   
22        # copy *.dat files into the subdir 
23        for f in files : 
24            if re.match ('^.*\.dat$', f) : 
25                dir.copy (f, "sftp://localhost/tmp/data/") 
26   
27      The above example covers most of the semantics of the filesystem package -- 
28      additional capabilities, such get_size() or move(), can be found in the 
29      individual class documentations. 
30  '''

41  '''Enforces an operation to continue even if the target entry does already

43  from such an operation.'''

47  is not given, the same operation would only apply to the given directory, and 
48  not to its children.'''

50  Dereference     =    4 
51  '''Enforces an operation to apply not to the entry pointed to by the target 
52  name, but to the link target of that entry – if that flag is not given, the 
53  same operation would apply to the entry directly, and its link target stays 
54  unaffected.''' 
55   
56  Create          =    8 
57  '''Allows a namespace entry to be created while opening it, if it does not 
58  already exist – if that flag is not given, the same open operation would cause 
59  a ’DoesNotExist’ exception. If the entry exists, the flag is ig- nored. This 
60  flag implies the ’Write’ flag.''' 
61   
62  Exclusive       =   16 
63  '''Implies a modification to the meaning of the Create flag: if the entry 
64  already exists, the Create flag is is no longer silently ignored, but causes an 
65  ’AlreadyExists’ exception.''' 
66   
67  Lock            =   32 
68  '''Enforces a lock on the name space entry when it is opened. Locks are 
69  advisory in SAGA, semantic details for locking are defined in the description 
70  of the open() call.''' 
71   
72  CreateParents   =   64 
73  '''An operation which would create a name space entry would normally fail if 
74  any path element in the targets name does not yet exist. If this flag is given, 
75  such an operation would not fail, but would imply that the missing path 
76  elements are created on the fly. This flag implies the ’Create’ flag.'''
