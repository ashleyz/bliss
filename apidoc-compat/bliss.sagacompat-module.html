Package sagacompat

source code

Bliss (BLiss IS SagaA) is a pragmatic, light-weight implementation of the OGF SAGA standard (GFD.90).

ATTENTION: The bliss.sagacompat module only exists to ensure compatibility with existing SAGA implementations (http://www.saga-project.org) and its use is highly discouraged. Please use the bliss.saga module (http://oweidner.github.com/bliss/apidoc/) instead!

More information can be found at: http://oweidner.github.com/bliss
